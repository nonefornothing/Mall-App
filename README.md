## General info
This project is simple Mall App with certain rules that have impact to total cost that program calculate.

Rules :
1. All clothes at any mall have same price (Rp. 100.000 / pcs)
2. All electronic at any mall have same price (Rp. 500.000 / pcs)
3. At kokas mall , each clothes have 20 % discount and each electronic have 10% discount.
4. At ambasador mall, each clothes have 15 % discount and each electronic have 25% discount.
5. At Pejaten Village mall, each clothes have 5 % discount if buy 1 clothes or 20% discount if buy more than 1 clothes and each electronic have 30% discount.
6. At Kemang village mall, each item (clothes and electronic) have 15% discount.
